Search LDAP, result in JSON format

Usage: /list/ldapentry/{ldapServer}/{baseDN}/{filter}

Refer standard ldapsearch command usage. An example of the search: 
~/list/ldapentry/ldap.corp.redhat.com/ou=users,dc=redhat,dc=com/(rhathiredate>=20171106000000Z)


Result: 
{ "New Employee List" : [ 
  {
    "rhathiredate" : "20171106080000Z", 
     "rhatweblogin" : "mei@mycorp.com", 
     "rhatgeo" : "NA",
     ... ...
}, {
... ...
}],
  "Total New" : 41,
  "LDAP Server" : "ldap.mycorp.com",
  "Base DN" : "ou=users,dc=mycorp,dc=com",
  "Filter" : "(rhathiredate>=20171106000000Z)"
}


Or: { "Error": "Cannot connect to the server" }
